# Registration Form Test Cases (Manual Testing)

## Project Description
This project contains manual test cases for validating the User Registration Form available at:

https://kirantestingacademy.com/form/

The purpose of this project is to verify whether the registration form works correctly when users enter valid and invalid data.

## Test Scenario
Verify the functionality of the User Registration Form.

## Fields in Registration Form
- User Name
- First Name
- Last Name
- Phone
- Email

## Test Coverage
The test cases cover the following:

- Field validation
- Mandatory field validation
- Valid input scenarios
- Invalid input scenarios
- Form submission functionality

## Expected Outcome
The system should accept valid data and successfully submit the registration form.  
If invalid data is entered, appropriate validation messages should be displayed.
